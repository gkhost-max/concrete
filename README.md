# concrete-paper-writing

Codex skill for writing, restructuring, polishing, and reviewing papers on concrete transport, pumping, extrusion, shotcrete, and sprayed concrete.

The skill is tuned for manuscripts targeting:

- Cement and Concrete Research (CCR)
- Cement and Concrete Composites (CCC)
- Construction and Building Materials (CBM)

## Install

Copy the whole `concrete-paper-writing/` folder into your Codex skills directory:

```bash
cp -R concrete-paper-writing ~/.codex/skills/
```

Then restart Codex.

## Use

Ask Codex to use `concrete-paper-writing` for tasks such as:

- drafting an abstract for a concrete pumping paper;
- restructuring a sprayed concrete manuscript for CBM;
- checking whether a paper is better suited to CCR, CCC, or CBM;
- auditing missing methods details for shotcrete rebound or pumpability experiments;
- polishing Chinese technical notes into journal-ready English.
